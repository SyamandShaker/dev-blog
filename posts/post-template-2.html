What is VS Code

Visual Studio Code (VS Code) is the name of the latest developer tool from Microsoft. By name, you might think it is the latest edition of Visual Studio, but VS Code is more like its flexible little brother: A lean code editor that supports software developers not only in editing, but also in managing and testing code. In particular, code files outside of projects can be quickly edited with VS Code without having to do without the usual support from an IDE. Not only the syntax is highlighted in color, but also other functions known from IDEs are provided. The highlight of VS Code is that it runs not only on Windows, but also on MacOS and Linux.

Visual Studio Code also has a lot to offer in the preview it is currently in. Especially C# and web developers can look forward to a lot of support from the editor. For other technologies will be provided in further updates. Here are some of the most important features:

  1. Tidy and fast editing of code at folder and file level without projects
  2. Fast control and navigation through fully adjustable key combinations and a command palette
  3. Syntax highlighting and bracket matching for a wide range of programming languages
  4. IntelliSense including icon navigation, refactoring and code preview for selected languages
  5. Powerful debugger for Node.js and C# applications
  6. Built-in Git support
  7. Text search in files and folders
  8. Automatic saving of files
  9. Extensive configurability and various included themes
